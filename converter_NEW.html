NTAG Southwest

DATA CONVERTER

Turn weekly Excel sheets into the app's data feed · runs entirely in your browser
Step 1 · Select Files

Drop your Excel sheets

Select one or many .xlsx files named like 2_15OCT25.xlsx. The converter reads the Annual Goals & Attn tab from each, pulls the date & sequence from the filename, and builds one clean JSON snapshot per file. Nothing leaves your computer.

Two drop zones: use Weekly for normal daily/weekly sheets. Use Month-End (EOM) for the snapshot taken on the last production day of a month — those become the official monthly records the Leaderboard reads from.

Step 2 · Convert

Build the data feed

Produces one YYYY-MM-DD.json per sheet plus an index.json manifest. Download them, then commit the whole set to the repo's data/ folder.

Step 3 · Download & Deploy

Your data feed is ready

This one file holds all weeks. Upload it to the repo root the same way you upload index.html — single file, no folders. This is the recommended path.

Optional — only if you're using GitHub Desktop later. Produces a data/ folder of separate weekly files.
